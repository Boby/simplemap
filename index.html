Software Versioning
*Items in grey have been completed

  1. Version 0.01 edit a map.
    *NOTE: map will only work on domains defined in Google API
  2. Version 0.02 drag and drop a valid geojson file
    *NOTE: map will only work on domains defined in Google API
  3. Version 0.03 combine edit and geojson combined into one file
  4. Use google maps as base layer. Draw/Edit feature is second layer.
  5. Changes to layer will be stored as GeoJSON and/or KML OpenLayers: http://openlayers.org/en/v3.5.0/apidoc/ol.format.GeoJSON.html#writeFeature Google Maps: https://developers.google.com/maps/documentation/javascript/datalayer
  6. User saves map, VALID geojson will be uploaded and saved to MySQL or postgreSQL.
  7. Table in database will have unique id
  8. ID in table will return to browser as a url.
  9. When user clicks url, a google map with the geojson data will open up
  10. Git versioning will be used for development.
